# blog

An blog system with 3 types of logged user:

	- Admin he can:
		-- create post,
		-- create comment
		-- manage all posts,
		-- manage all comments,
		-- manage users (delets or change user types: bloger/reader)

	- Bloger he can:
		-- create posts,
		-- create comments,
		-- manage his posts,
		-- manage his comments

	- Reader he can:
		-- create comments,
		-- manage his comments
